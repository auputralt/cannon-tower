# 🏰 Cannon Keep — Tower Defense

A self-contained, browser-based tower defense game built entirely in a single HTML file. No frameworks, no build steps — just open and play.

## 🎮 Play Now

**Live demo:** [Play Cannon Keep](https://mrputra.github.io/cannon-tower/)

Just open `cannon-tower-defense.html` in any modern browser — it works offline too.

## 📸 Features

- **Canvas-based rendering** — smooth 60fps gameplay
- **Multiple tower types** — strategically place different towers to defend your keep
- **Wave-based enemies** — increasingly challenging waves of foes
- **Mobile-friendly** — touch controls with responsive layout
- **Zero dependencies** — single HTML file, works offline
- **No server needed** — runs entirely in the browser

## 🕹️ How to Play

1. **Open the game** in your browser
2. **Tap/click** on the grid to place towers
3. **Defend your keep** from incoming waves of enemies
4. **Earn gold** by defeating enemies to build more towers
5. **Survive** as long as you can!

## 🛠️ Tech

- Pure HTML5 Canvas + JavaScript
- No external libraries or frameworks
- Mobile-first responsive design
- Works on any modern browser (Chrome, Safari, Firefox, Edge)
