## 🎮 Play

Open [cannon-tower-defense.html](cannon-tower-defense.html) in any modern browser — it works offline too. No server, no install, no nonsense.

**Live demo:** [cannon-keep.netlify.app](https://cannon-keep.netlify.app)

---

## 🕹️ How to Play

1. **Choose a map** — Meadow Run, Dusty Dunes, or Frostpeak Pass
2. **Tap a pad** (◯) and choose a tower to place:
   - 💣 **Cannon** — splash damage, slow fire rate
   - 🏹 **Ballista** — rapid fire, single target
   - ❄️ **Frost** — slows enemies in range
3. **Tap a placed tower** to upgrade its power or sell it back
4. **Start each wave** and defend your keep from incoming enemies
5. **Earn gold** by defeating enemies — spend it on more towers
6. **Survive all waves** across 3 maps to conquer Cannon Keep!

## 🏗️ Features

| Feature | Details |
|---------|---------|
| 🎯 **3 Tower Types** | Cannon (splash), Ballista (rapid), Frost (slow) |
| 🗺️ **3 Maps** | Meadow Run · Dusty Dunes · Frostpeak Pass |
| 👾 **4 Enemy Types** | Scouts, Runners, Tanks, Bosses — each with unique stats |
| ⬆️ **Tower Upgrades** | 2 upgrade levels per tower for more power |
| 💰 **Economy System** | Earn gold, strategically build and sell towers |
| ⚡ **Speed Control** | 1× / 2× game speed toggle |
| 🏆 **Star Ratings** | Earn stars based on performance |
| 📱 **Mobile-Friendly** | Touch controls, responsive layout |
| 🔇 **Sound Effects** | Procedural audio for shots, hits, and builds |
| 🌐 **Works Offline** | Zero dependencies, single HTML file |
| 🖥️ **Any Browser** | Chrome, Safari, Firefox, Edge — desktop & mobile |

## 🤝 Contributing

Contributions are welcome! Whether it's a bug fix, new feature, or a fresh coat of paint — all ideas are appreciated.

### Quick Start

1. **Fork** this repo
2. Clone your fork and open `cannon-tower-defense.html` in a browser
3. Edit, save, and refresh — no build step needed
4. Test your changes thoroughly
5. [Open a Pull Request](https://github.com/auputralt/cannon-tower/pulls)

### Ideas for Contributions

- 🗺️ **New maps** with unique layouts and wave designs
- 🏗️ **New tower types** (e.g., Lightning, Poison, Arcane)
- 👾 **New enemy types** with special abilities
- 🎨 **Visual polish** — animations, particle effects, screen shake
- 🔊 **Better audio** — richer sound effects or background music
- 📱 **UX improvements** — tutorials, tooltips, settings menu
- 🏆 **Leaderboard** — local high score tracking
- 🌍 **Localization** — translate the game into other languages
- ♿ **Accessibility** — keyboard controls, screen reader support

### Contribution Guidelines

- Keep it as a **single HTML file** — that's the project's charm
- Test on **both desktop and mobile** browsers
- Write **clean, readable code** with comments
- Keep the **file size reasonable** — no external assets unless necessary
- Be respectful and constructive in issues and PRs

### Reporting Bugs

Found a bug? [Open an issue](https://github.com/auputralt/cannon-tower/issues) with:
- A description of the problem
- Steps to reproduce
- Browser and device info
- Screenshots if possible
